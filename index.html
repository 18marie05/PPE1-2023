Miniprojet

PPE1 2023 - Création de site

Sujet

Récupération d'URLs afin d'en faire un tableau html.

En quoi consiste le miniprojet ?

On a procédé de la manière suivante :

  1. Utilisation d'une liste d'URL dans un fichier
  2. Récupérer les URLs à partir d'un script
  3. Récupérer le numéro de ligne ou d'URL, le code HTTP (le corriger si possible), l'encodage
  4. Tout transformer en tableau HTML une fois le script finalisé

Et après ?

Cette étape est la dernière : on donne le lien du tableau HTML pour le visualiser sur ce site
